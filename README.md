## Run Locally

Clone the repo

```bash
    git clone https://github.com/ashwini-chaudhar/portfolio-app
```

Go to the project directory

```bash
  cd portfolio-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
